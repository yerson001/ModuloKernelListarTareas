# ModuloKernelListarTareas

Compilar:
~~~
1- Ejecutar "make"
~~~
Insertar Modulo para listar tareas:
~~~
2- En modo ROOT "insmod Linear.ko"
~~~
Ejecutar 
~~~
3- usar :  dmesg
~~~

Eliminar Modulo listar tareas:
~~~
4- Ejecutar el comando "rmmod ListTasksLinear"
~~~

Repetir paso para el DFS
